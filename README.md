# Space Portfolio

A personal portfolio website built with **Next.js 16**, **React 19**, **TypeScript**, **Tailwind CSS**, and **Three.js**.

### Skills

- **Languages:** Python, C, C++, Java, JavaScript, SQL
- **Data & ML:** Pandas, NumPy, Scikit-learn, Apache Kafka, Apache Spark, Jupyter
- **Tools:** Git, Docker

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **UI:** React 19, Tailwind CSS 3, Framer Motion
- **3D:** Three.js via @react-three/fiber & @react-three/drei
- **Deployment:** Netlify

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
npm run build
```
